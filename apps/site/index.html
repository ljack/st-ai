Quick Start

Get up and running with the Sports Tracker CLI and connect it directly to your agent.

Use this if you want Sports Tracker as a skill for Claude Code, OpenClaw, or Gemini.

# 1) Install the CLI package globally
npm install -g @st-ai/cli

# 2) Perform initial interactive authentication
st login

# 3) Register the Sports Tracker skill to your agent
npx skills add ljack/st-ai

How It Works

1. Direct Secure Login Authenticate once directly against Sports Tracker. Your credentials are used locally to generate secure TOTP tokens, obtaining a short-lived session key.
2. Secure Local Storage Your session key is stored locally in `~/.config/st-ai/config.json` with secure `0600` permissions. No passwords or tokens ever leave your machine.
3. AI-Ready Outputs Retrieve active metrics, routes, calendar logs, and GPX files using standard format commands. Output cleanly structured JSON data directly into your LLM prompt context.

What can I do with st.ai?

Run commands interactively or let your agent execute them programmatically to monitor your progress.

Login & Setup

Authenticate and verify your active session credentials.

st login

List workouts

Get a quick history list of your recent fitness activities.

st workouts

Summary Stats

Show total stats and aggregate values by activity type.

st stats

Export GPX / FIT

Download workout route files to import into Strava or GPX maps.

st export [key] [outPath]

Sync & Upload

Upload pre-built SML/JSON files to add new training logs.

st upload [smlPath]

Delete entries

Remove incorrect workout logs from the Sports Tracker database.

st delete [key]

FAQ

Where do my credentials live?

Your credentials remain 100% local. Once verified, the login key obtains a session key, which is saved locally under your profile settings. We never run a backend database or save user info.

Do I need a backend?

No backend is needed. It runs fully local as a CLI client, calling the Sports Tracker API directly on your behalf.

Is this officially endorsed?

No. This is a community-developed, open-source tool created by a developer who wanted scriptable and agentic access to their fitness data.

Can AI agents read my GPX routes?

Yes! With the `--json` option, the CLI yields structured output. AI agents can read this, analyze your training frequency, plot route summaries, and give suggestions.
